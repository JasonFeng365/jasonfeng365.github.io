A substring is contained within another string if letters can be removed from the bigger string, without changing its order, to obtain the substring.
For example, the substring "jvjv" is contained by the string "javajavajava", because letters can be removed from "javajavajava" to produce the substring "jvjv". "jvaaaaa" is also contained, but "jvaaaaaa" (one extra 'a') is not.
"json" is contained by "jason", but "josn" is not, because it is not in the order.

You are given one huge 100,000 character string, and 1,000 other strings of lengths up to 100. The first line of input will be the huge string, and the next 1,000 lines will be smaller strings.
Determine how many of those smaller strings are contained by the large string.

Example input:
abcdefg
aeg
bb
a
adcg
efg
In this case, aeg, a, and efg are contained, but bb and adcg are not. Therefore, the answer will be 3.
