Maze Filling

Scientists are doing experiments on how fast mice can solve mazes! Given a maze, we need to patch any holes.

A maze, represented by a 2D grid, is given. A # represents a wall, and a . represents an empty tile. Before the maze can be cleaned, the holes in the maze must be patched.
A hole is an empty tile that is surrounded on all four sides will a wall tile. For example: 


########.#
#...##.###
##.##.#.##
##########

In this grid, there are 8 spaces, but only 3 of them are holes. The four spaces on the right do not count, as each hole is not surrounded by a wall on all sides. The space on the top row also does not count, because it does not have a wall above it. Only the three spaces in the 6th, 7th, and 8th columns are valid holes. Thus, the answer would be 3.
