Integral

Featured on SCPE Fall 2023

Many contests will have problems that are purely implementation. Problems like these provide the entire algorithm, but it is up to the programmer to implement a correct solution in a fast time.
A good example of this is Problem G: Fading Wind from the Spring 2023 PacNW ICPC. There, the algorithm is given, and the programmer must implement an accurate solution while making sure to follow the algorithm exactly as specified.
SCPE Integral and Riemann Sum are both this type of problem. You don't need to know any calculus at all to solve this- the algorithm is completely provided!
