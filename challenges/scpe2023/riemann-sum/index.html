Riemann Sum

Featured on SCPE Fall 2023

In addition to being a pure implementation problem like Integral, this problem shows some rounding errors form floating-point numbers. I was lucky that Lego was able to catch this bug before I put the problem on the contest- the testcases initially had this rounding error too!
Be careful with the for loop on this problem. The challenge review shows a few ways to circumvent the rounding errors.
